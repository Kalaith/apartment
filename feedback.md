# Project Review

## Project

**Name:** Apartment Manager

**One Sentence Pitch:** Manage an apartment complex, keep your tenants happy.


# 1. First Impression
Spend 10-20 minutes using the project without making changes.
- UI is dated and needs a overhaul
- Money is to easy to earn, there is no reason to sell of an apartment
- hiring staff made no notiable impact, expected that with everyone wouldn't need to do maintaince, maybe even manager handling animal approvals for happiness.
- I didn't feel any need to reject a tenant


# 2. Story
Would I keep reading?
- this is more of a gameplay game then story, but there is tenant being friends with eachother, id like to get a emergent story via tenant interactions, e.g sarah pays alot in rent but makes other tenants unhappy, do you kick her, or let the other tenants leave

# 3. Does it Have Potential?
Ignoring its current state...
Could this become something great with more work?
Yes, I think with a slower curve and better UI, this game would be playable as it is.

# 4. Biggest Problems
- to easy to make money
- dated UI

# 5. Best Parts
What already works well?
- does feel like your managing apartments, doing the upgrades and repairs

# 6. Next Goal
What is the **single most important milestone**?
- improve the dated UI, no overlaping elements

# 7. Content Notes (Optional)
Would this make an interesting YouTube video or livestream?
I think it could be interesting if the emergent tenant stories worked.
